# Student-Performance-Analysis-Using-Python-Tableau

A beginner-friendly data analytics project that analyzes student academic performance using Python, Pandas, and Tableau. The project demonstrates how raw student data can be cleaned, transformed, analyzed, and visualized to generate meaningful insights about academic performance, study behavior, and attendance impact.

This project focuses on performance analysis, study habits, attendance impact, and data-driven visualization dashboards using Tableau.

# Technologies Used
• Python
• Pandas
• NumPy
• Matplotlib
• Seaborn
• Tableau
• CSV Dataset

# Flow of Project 
→ Feature Handling → Statistical Analysis & Aggregations → Data Visualization (Matplotlib & Seaborn) → Data Export (CSV) → Tableau Dashboard Creation → Student Performance Insight Reporting

# Project Features
  # 1. Data Extraction
  • Imported student dataset from CSV file
  • Dataset includes:
    • Student Gender
    • Study Hours per Day
    • Attendance Percentage
    • Math Score
    • Science Score
    • Parent Education
    • Result (Pass/Fail)
  
  # 2. Data Cleaning & Preparation
  • Handled missing values using mean (numerical) and mode (categorical)
  • Removed duplicate records
  • Ensured correct data types for all columns
  • Standardized dataset for analysis
  
  # 3. Feature Handling & Outlier Detection
  • Applied IQR method to detect outliers in scores
  • Identified extreme performance values
  • Prepared clean dataset for accurate analysis
  
  # 4. Statistical Analysis
  • Performed descriptive statistics on scores
  • Analyzed:
    • Mean, Median, Min, Max
    • Score distribution
    • Compared performance across categories
  
  # 5. Business (Academic) Analytics
  • Gender-wise Performance Analysis
  • Result-wise Attendance Analysis
  • Study Hours vs Score Relationship
  • Identified factors affecting student performance
  
  # 6. Data Visualization (Matplotlib & Seaborn)
  • Generated multiple plots including:
    • Histogram (Score Distribution)
    • Scatter Plot (Study Hours vs Score)
    • Box Plot (Outlier Detection)
    • Bar Charts (Category Comparison)
    • Heatmap (Correlation Analysis)
    • Violin & Count Plots
    • Saved all visualizations into a structured folder (visualization/)
  
  # 7. Data Export for Visualization Tools
  • Cleaned dataset exported to CSV format
  • File used for dashboard creation in Tableau
  
  # 8. Tableau Dashboard Integration
  • Imported dataset into Tableau
  • Created interactive dashboards including:
    • Student Performance Overview
    • Study Hours vs Score Analysis
    • Gender-wise Comparison
    • Attendance Impact Dashboard
    • Distribution & Outlier Analysis
  
  # 9. Insight Generation
  
  • The analysis provided insights such as:
  • Students with higher study hours tend to score better
  • Attendance has a strong impact on academic performance
  • Performance varies across different categories (gender, parent education)
  • Presence of outliers in student scores
  • Strong correlation between study habits and results
  
  These insights help understand student behavior and improve academic performance strategies.

# Project Outcome

This project demonstrates a complete data analytics workflow including data cleaning, statistical analysis, visualization, and dashboard development. It showcases the ability to transform raw data into meaningful insights and build interactive dashboards for data-driven decision making.
